Browse the Album Files

Set the Working Directory

When you start the Album EPUB Browser, you see a file list screen similar to the Explorer. The default working directory is the "Album Epubs" subdirectory under the "My Documents" folder. You can either put the .album.epub files in it directly, or click the "..." button to browse another directory, as shown below.

This software displays the information of the albums in the working directory, and you can still manage the files in this directory through Windows Explorer. As shown below, you can switch the display modes via the button "List View" and "Icon View". In the list view, the complete information of the album files are listed. While in icon view, the thumbnail of the cover of the album are displayed. In the both mode, when you click an album file in the list, its cover and the complete information are displayed in the right column.

Sorting and Searching

Similar to Windows Explorer, this software also allows you to sort the album files in the working directory, not only by file name, but also by album title, author, or dates in ascending or descending order. That makes it easy to arrange or find the albums.
