Overview

This is a reader developed for the album EPUB files produced by Album EPUB Studio. Although Album EPUB Studio produces standard EPUB files that can be opened by any reader software or hardware. Album EPUB Browser can open the files much faster, even if the sizes of them are hundreds of MB or more than GB. This is because Album EPUB Browser recognizes the exclusive information that Album EPUB Studio has embedded in these EPUB album files, so it doesn't need to unzip the whole album file to retrieve the part of the file that the user wants to read to display it. Furthermore, the software can search for photos in multiple album files so that the user doesn't have to click on the photos one by one. The realization of this search function depends on the fact that the user has properly labeled the photos with the date, location and a short description when creating the album files with Album EPUB Studio.

Album EPUB Browser has three main functions: First, it lists the EPUB albums in the selected working directory. Windows Explorer basically only lists the file name, file date, and file size of album files. This software can list the cover, album title, author, and dates of the albums in the working directory. Second, this software can open the album files quickly and download the photos. Third, you can search photos in the album files in the working directory (excluding subdirectories) via this software. You select the shooting date range, locations, or some keyword of the photo-description text, then it picks out the photo s in the album files, and you can view and download the photos directly in the search results.
