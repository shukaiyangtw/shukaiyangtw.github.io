Modify a Photo

The software is not an imaging application, but it also provides several functions for individual photo, including rotation and album-cover making. In the photo set, (1) pick a photo then (2) click the toolbar button Edit the Photo Info., or just double click the photo to be modified.

Double clicking on a photo, a dialog box pops as shown in the figure, and provides the functions: (a) Label the photo with a short text, then click Done. (2) Replace the photo by another image file without reordering the photo set. (3) Set the photo as the cover image by button Set the Album Cover. (3) Rotate the photo by 90 degrees. (5) Export the photo to an image file by button Save As.

The label text of the photo shows both on the photo set and the preview.
