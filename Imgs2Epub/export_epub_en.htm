Exportation and Importation

The user can export an album project to an EPUB file anytime, no matter it is finished or not. Creation of some albums spends months, such as diaries or experiment records. As a result or as a backup, it is worthy of appreciation to backup albums at all times. If you want to continue editing some finished albums, just import them.

Export to an EPUB File or Website

To export an album project, first (1) pick the project in the list, then (2) click the toolbar button Export to a File. The file dialog box shows to select the file name and path. (3) Select the file type for exportation, option Epub album files (*.epub) exports the project to an .album.epub file, and option Zipped HTML files (*.zip) exports the project to an archive of web pages you have seen in the preview window.

Take the freeware Calibre for example, the exported .album.epub file is read like this. When an EPUB file is loaded in a reader, the typesetting is automatically changed according the dimension of the reader, and the thumbnails of photos are clickable. Take Calibre for example, once you (1) click on a thumbnail, it shows an annotation about the photo. (2) There is an arrow for expansion. Click the arrow, then you get the full-screen photo image. You can also (3) right-click on the photo, and you get the menu item View image to get the original photo file.

If you choose to export an album project to a zip of web pages. When you unzip the exported file, put the extracted files on a web site or open index.html on the local machine. You can see the same pages with the preview. Each thumbnail is clickable.

Import an Album EPUB File

To continue editing a finished album, (1) click the toolbar button Import an .album.epub File, then pick a file by the file dialog box. Or just (1) drag the file into the list of album projects. It shows in the list and become editable again.

If the album is finished and exported to an EPUB file, click the toolbar button Delete the Album to release the disk-space occupied by the project.
