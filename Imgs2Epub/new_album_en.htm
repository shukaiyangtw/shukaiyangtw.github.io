Create a New Album

Sometimes arranging photos spends days. It is more than editing a file in hours. So the software treat the albums as projects. As shown below, once you create a new album, it is a project in the list, then you can edit it anytime. There is no Save button because all your modifications will be saved automatically, and you can preview the album by the toolbar button Preview the Album. When your album is finished, export it to an EPUB file or web pages by the toolbar button Export to a File. The project remains unless you delete it by button Delete the Album. The exported EPUB file can be imported again.

New an Album Project

To build a new album, first (1) click the toolbar button New Album Project, then an empty project (Untitled Album) is listed.

Let's enter the new album project. (2) Click the cover of album (Untitled Album) then (3) click button Edit the Album Project, or just (2) double click on the album cover, it enters the editable album-info window.

Fill the Album Info.

Double clicking on the album in the figure above, you enter the album-info window shown below. In the left is the blank image of album cover, followed by the optional text fields of the basic information of this album including (1) the title, (2) the name of the album author, (3) main locations of photography, and (4) the first and the last date when the photos are taken. (Only the first date has to be set, the last date will be updated automatically when more photos are added.) In the right (5) is the text field for a brief story or introduction to the album.

In the window, there is no Save button, because anything you edited will be saved in the project automatically. Besides, there is a toolbar button Preview the Album for you to preview the results as web pages. An EPUB book basically consists of web pages. As shown below, the information you edited is converted to the preface of the book.

When a new album project is created, an empty chapter titled New Chapter is also created. It is already listed in the table of contents. This default title is usually unwanted, so let's change it.

Edit the Title of a New Chapter

In the table of contents, first (1) click the chapter to be renamed, (2) click the toolbar button Rename the Chapter, then the dialog box shows. (3) Key in the new title of the chapter, you may also (4) rename the sub-directory optionally, then (5) click the OK button.

As mentioned, an EPUB is basically a collection of web pages, and this software places all the photos of a chapter in its own sub-directory. Changing the sub-directory names (4) has no effect on the result EPUB file. If you will export the album to web pages, the sub-directory names will affect the URLs of photos.

Add or Delete Other Chapters

Although you can put all photos of the album in a single chapter, it is better to divide a lot of photos into multiple chapters. It helps to accelerate the loading of the result EPUB book or web pages. To create another chapter, simply (1) click the toolbar button Add a New Chapter. As shown below, the new chapter is listed in the table of contents, then you can also rename it. (the sub-directory name is assigned ch002 by default).

When there are two or more chapters, you can (2) reorder the chapters by dragging. If you want to delete one of them, click it on the table contents, then click toolbar button Delete the Chapter.
